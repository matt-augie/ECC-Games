# The Wheel — Edina CC Golf App

A Progressive Web App (PWA) for tracking The Wheel and 9 Point Game at Edina Country Club.

## Games included
- **The Wheel** (5 players): 3 two-on-two matches, 5·5·5 betting format, Greenies, auto-press
- **9 Point Game** (3 players): Net scoring, –/pt with player presses, Greenies

## Deploy to GitHub Pages (free hosting)

1. Create a new GitHub repository (e.g. )
2. Upload all files in this zip to the repo root
3. Go to **Settings → Pages → Source → main branch → / (root)** → Save
4. Your app will be live at 

## Install on iPhone

1. Open the GitHub Pages URL in **Safari**
2. Tap the **Share** button (box with arrow)
3. Tap **Add to Home Screen**
4. Tap **Add** — it will appear as a full-screen app icon
